# Defra Legacy Reverse Engineering Plugin

This plugin assists engineers working on Defra's Legacy Application Programme (LAP).

## Context

Defra maintains a large estate of legacy applications built across decades using a variety of technologies. The LAP programme aims to understand, document, and modernise these systems.

## Conventions

- Use British English in all generated documentation
- Refer to the programme as "LAP" or "Legacy Application Programme"
- Treat all legacy code as potentially undocumented — never assume documentation exists
- When creating a new skill, add an entry to the Skills table in README.md
- When creating a new agent, add an entry to the Agents table in README.md
